PASSO 1: Fazer Login
   → Abra: http://localhost:5000/login
   → Email: admin@escola.com
   → Senha: Mudar@123
   → Clique: Entrar

PASSO 2: Explorar Dashboard
   → Você verá o menu principal
   → Clique em qualquer opção que aparecer

PASSO 3: CRIAR DADOS (na ordem recomendada)

   A. Criar Setor (opcional, mas bom ter)
      → Admin Dashboard
      → Clique em "Gerenciar Setores"
      → Clique em "+ Novo Setor"
      → Ex: "Administrativo", "Pedagógico"

   B. Criar Curso
      → Admin Dashboard
      → Clique em "Gerenciar Cursos"
      → Clique em "+ Novo Curso"
      → Ex: "Python", "R$500", "3 meses"

   C. Criar Módulo
      → Admin Dashboard
      → Clique em "Gerenciar Módulos"
      → Clique em "+ Novo Módulo"
      → Selecione o curso criado
      → Ex: "Variáveis e Tipos", "20 horas"

   D. Criar Professor
      → Admin Dashboard
      → Clique em "Gerenciar Funcionários" (ver professores)
      → OU na página principal: "Criar Professor"
      → Preencha dados
      → Email e senha gerados automaticamente

   E. Criar Funcionário
      → Admin Dashboard
      → Clique em "Gerenciar Funcionários"
      → Clique em "+ Novo Funcionário"
      → Selecione setor
      → Senha gerada automaticamente

   F. Criar Turma
      → Na página principal: "Gerenciar Turmas"
      → Clique em "+ Nova Turma"
      → Selecione curso e professor criados
      → Defina horário, sala, capacidade

   G. Cadastrar Alunos
      → Na página principal: "Gerenciar Alunos"
      → Clique em "+ Novo Aluno"
      → Preencha dados
      → Email e senha definida por você

   H. Registrar Frequência
      → Na página principal: "Registrar Frequência"
      → Selecione turma e data
      → Marque presença

   I. Lançar Notas
      → Na página principal: "Lançar Notas"
      → Selecione turma, aluno, módulo
      → Digite 3 notas
      → Média calculada automaticamente

   J. Gerar Relatórios
      → Na página principal: "Relatórios"
      → Clique no tipo desejado
      → Veja dados em tempo real
