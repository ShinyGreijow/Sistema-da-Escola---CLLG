                          🎮 COMO USAR O SISTEMA
════════════════════════════════════════════════════════════════════════════════

1. ACESSE:
   http://localhost:5000/login

2. FAÇA LOGIN COMO ADMIN:
   admin@escola.com / Mudar@123

3. CLIQUE EM DASHBOARD:
   Veja os gráficos e estatísticas

4. TESTE CADA MÓDULO:
   
   PEDAGÓGICO:
   → Clique em "Pedagógico" no dashboard
   → Veja lista de turmas
   → Clique em uma turma
   → Clique em um aluno
   → Veja dados completos com notas, frequência, status, relatórios
   → Adicione novo relatório
   → Altere status do aluno

   COMERCIAL:
   → Clique em "Comercial" no dashboard
   → Veja KPIs
   → Clique em "Nova Matrícula"
   → Selecione aluno e turma
   → Sistema calcula valor automaticamente
   → Escolha quantidade de parcelas
   → Sistema cria parcelas automaticamente
   → Na aba Financeiro, registre pagamentos

   ANEXOS:
   → Clique em "Alunos"
   → Clique em um aluno
   → Clique em "Anexos"
   → Faça upload de arquivo
   → Escolha tipo (foto, documento, contrato)

   RELATÓRIOS:
   → Clique em "Relatórios"
   → Escolha tipo (Turmas, Frequência, Desempenho)
   → Veja dados completos
