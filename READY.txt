════════════════════════════════════════════════════════════════════════════════
                    🛠️ COMANDOS DOCKER PARA MANUTENÇÃO
════════════════════════════════════════════════════════════════════════════════

Ver status do container:
  docker ps

Ver logs em tempo real:
  docker logs sistema_escola -f

Reiniciar aplicação:
  docker restart sistema_escola

Parar aplicação:
  docker-compose -f tmp/school_system/docker-compose.yml down

Iniciar aplicação:
  docker-compose -f tmp/school_system/docker-compose.yml up -d

Acessar terminal do container:
  docker exec -it sistema_escola sh

Rebuild (se fizer mudanças):
  docker-compose -f tmp/school_system/docker-compose.yml build --no-cache
