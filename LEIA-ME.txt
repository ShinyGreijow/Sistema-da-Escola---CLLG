🎯 MENU ADMIN RÁPIDO
====================================================================

Dentro do sistema, você tem acesso direto a:

1. Dashboard Admin (/admin)
   → Visualiza: Total alunos, professores, turmas, cursos, receita
   → Últimas matrículas registradas

2. Funcionários (/admin/funcionarios)
   → Criar novo funcionário
   → Editar dados
   → Atribuir setor
   → Gerar senha temporária automaticamente

3. Cursos (/admin/cursos)
   → Criar novo curso
   → Editar valores
   → Definir duração
   → Ativar/desativar

4. Módulos (/admin/modulos)
   → Criar módulos
   → Associar a cursos
   → Definir carga horária
   → Ordenar sequencialmente

5. Setores (/admin/setores)
   → Criar setores
   → Organizar funcionários
   → Ver total por setor

6. Usuários (/admin/usuarios)
   → Listar todos
   → Ativar/desativar contas
   → Ver tipo de usuário

7. Alunos (integrado)
   → Listar alunos
   → Ver detalhes completos
   → Acompanhar desempenho

8. Turmas (integrado)
   → Criar turmas
   → Atribuir professor/curso
   → Definir horários e salas

9. Frequência (integrado)
   → Registrar presença
   → Ver relatórios

10. Notas (integrado)
    → Lançar 3 notas por módulo
    → Cálculo automático de média

11. Relatórios
    → Mapa de Turmas (ocupação)
    → Frequência (percentual por aluno)
    → Desempenho (aprovação/reprovação)

🔧 COMANDOS ÚTEIS DOCKER
====================================================================

Ver logs da aplicação:
  docker logs sistema_escola -f

Acessar container:
  docker exec -it sistema_escola sh

Parar aplicação:
  docker compose -f tmp/school_system/docker-compose.yml down

Reiniciar:
  docker restart sistema_escola

Rebuild:
  docker compose -f tmp/school_system/docker-compose.yml build --no-cache
