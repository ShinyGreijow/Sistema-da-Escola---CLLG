                         🎯 TESTE OS FLUXOS PRINCIPAIS
════════════════════════════════════════════════════════════════════════════════

FLUXO 1: Criar Novo Aluno e Gerar Parcelas
   1. Login como admin@escola.com
   2. Vá em Alunos > Novo Aluno Completo
   3. Preencha dados (nome, email, senha, telefone, etc)
   4. Selecione um curso
   5. Escolha 3 parcelas
   6. Sistema cria automaticamente:
      - Aluno
      - Usuário de login
      - Status "Não Iniciou"
      - Matrícula na turma
      - 3 parcelas automáticas
   7. Acesse /comercial/parcelas/<id> para ver as parcelas

FLUXO 2: Registrar Pagamento
   1. Vá em Comercial > Financeiro
   2. Clique em Ver Parcelas
   3. Selecione método de pagamento
   4. Clique em "Registrar Pag."
   5. Sistema marca como pago e gera data
   6. Gere comprovante

FLUXO 3: Consultar Histórico
   1. Acesse /comercial/historico/<id_aluno>
   2. Veja todas as matrículas
   3. Veja todos os pagamentos
   4. Veja status financeiro

FLUXO 4: Pedagógico
   1. Login como professor
   2. Vá em Pedagógico
   3. Selecione uma turma
   4. Clique em Frequência
   5. Registre presença/falta
   6. Clique em Notas
   7. Registre notas (cálculo de média automático)
