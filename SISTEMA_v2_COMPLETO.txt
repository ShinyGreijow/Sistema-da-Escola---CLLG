                         ⚙️ COMO IMPLEMENTAR v2.0
════════════════════════════════════════════════════════════════════════════════

PASSO 1: Backup (AGORA!)
   └─ Copie todo o projeto para Desktop/Google Drive
   └─ Use os scripts: backup.bat

PASSO 2: Atualizar banco de dados
   └─ Execute: python database_v2.py
   └─ Isso cria as novas tabelas sem perder os dados antigos

PASSO 3: Copiar novo app
   └─ Renomeie: app.py → app_old.py (backup)
   └─ Renomeie: app_v2.py → app.py
   └─ Copie para container: docker cp app.py sistema_escola:/app/

PASSO 4: Criar templates novos
   └─ Copie: templates/dashboard_v2.html
   └─ Copie: templates/pedagogico/
   └─ Copie: templates/comercial/
   └─ Copie: templates/relatorios/
   └─ Copie: templates/alunos/anexos.html

PASSO 5: Reiniciar container
   └─ docker restart sistema_escola

PASSO 6: Acessar
   └─ http://localhost:5000/login
   └─ Email: admin@escola.com
   └─ Senha: Mudar@123


════════════════════════════════════════════════════════════════════════════════
                       📋 ROTAS E FUNCIONALIDADES
════════════════════════════════════════════════════════════════════════════════

DASHBOARD:
   GET  /dashboard                      → Dashboard v2.0 com gráficos

PEDAGÓGICO:
   GET  /pedagogico                     → Dashboard pedagógico
   GET  /pedagogico/aluno/<id>          → Detalhes completo do aluno
   POST /pedagogico/aluno/<id>/relatorio/novo    → Criar relatório
   POST /pedagogico/aluno/<id>/status/novo       → Alterar status

COMERCIAL:
   GET  /comercial                      → Dashboard comercial
   GET  /comercial/matricula/nova       → Formulário nova matrícula
   POST /comercial/matricula/nova       → Criar matrícula
   GET  /comercial/financeiro           → Listar financeiros
   GET  /comercial/parcelas/<id>        → Ver parcelas
   POST /comercial/parcela/<id>/pagar   → Registrar pagamento

ANEXOS:
   GET  /aluno/<id>/anexos              → Ver anexos do aluno
   POST /aluno/<id>/anexo/novo          → Upload novo anexo

RELATÓRIOS:
   GET  /relatorios/historico/<id>      → Histórico completo com auditoria
